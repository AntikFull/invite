Как получить API ключ?​

Включаем V*N Канада или США (у меня получилось с V*N Satoshi Канада)

Чтобы получить API ключ нам необходимо перейти по ссылке:

https://aistudio.google.com/app/apikey 

И нажать на соответствующую кнопку ⁡Create API key.

После нажатия на кнопку, нам нужно будет выбрать приложение ⁡Generative Language Client.

Нажимаем ⁡Create API key in existing project⁡ и ожидаем появления окна с генерацией вашего ключа.

Спустя пару секунд у нас появится окно уже с созданным API ключом. Мы можем скопировать его в один клик, нажав на кнопку ⁡Copy.
